# REVPAY – Digital Payment & Money Management System

REVPAY is a backend-focused, monolithic financial web application built using **Java Spring Boot** and **MySQL**.  
The system enables secure digital payments and money management for both **personal** and **business** users, providing features such as wallet management, transactions, invoices, loans, notifications, and analytics.
